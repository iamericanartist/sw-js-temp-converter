Temperature Converter

Instructions

Write a program that will convert a temperature from fahrenheit to celsius, or from celsius to fahrenheit. In the HTML, have one input field where someone can enter in a temperature.
Add an event handler to the input field that checks if the user pressed the enter key, and if that happens, perform the conversion.
